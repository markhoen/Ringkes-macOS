# RINGKES 🖥️📦

**RINGKES** adalah aplikasi desktop berbasis macOS yang dirancang khusus untuk mengompresi file PDF secara cepat, aman, dan efisien. Diambil dari bahasa Jawa yang berarti ringkas, padat, dan praktis, aplikasi ini hadir sebagai solusi untuk meringankan beban ruang digital Anda tanpa mengurangi esensi penting di dalamnya.

> **"Cilik Ukurane, Gede Manfaate."**
> *(Kecil ukurannya, besar manfaatnya).* > Sebuah filosofi luhur Jawa yang melandasi pembuatan aplikasi ini—membuktikan bahwa efisiensi dan ukuran yang ringkas justru menyimpan kekuatan dan kegunaan yang luar biasa.

---

## ✨ Fitur Utama

* **Kompresi Instan:** Perkecil ukuran file PDF Anda dalam hitungan detik dengan algoritma yang optimal.
* **Drag and Drop:** Cukup seret dan lepas file PDF Anda ke dalam aplikasi untuk langsung memulai proses.
* **Kualitas Terjaga:** Mengurangi ukuran file (file size) secara signifikan namun tetap mempertahankan keterbacaan teks dan dokumen.
* **Keamanan Lokal:** Proses kompresi dilakukan sepenuhnya secara *offline* di perangkat Anda, menjaga kerahasiaan data penting Anda.
* **Desain Minimalis:** Antarmuka yang bersih, modern, dan selaras dengan ekosistem macOS.

---

## 🎨 Filosofi Logo

Ikon aplikasi RINGKES memadukan elemen **Blangkon Jawa** dan **Tumpukan Dokumen**. Blangkon melambangkan kesiapan, kebijaksanaan, dan identitas budaya lokal yang melandasi proyek ini, sementara tumpukan dokumen dengan panah ke bawah secara visual merefleksikan fungsi utama aplikasi: merapatkan dan meringkas data digital.

## 🚀 Cara Instalasi & Penggunaan

### Kebutuhan Sistem
* macOS 14.0 (Sonoma) atau versi yang lebih baru.

### Cara Penggunaan
1. Unduh rilis terbaru aplikasi di halaman **Releases**.
2. Buka berkas `.dmg` atau jalankan aplikasi **RINGKES**.
3. Tarik (*drag*) file PDF yang ingin Anda kompres, lalu letakkan (*drop*) ke dalam jendela aplikasi.
4. Tunggu beberapa saat, dan file PDF Anda yang baru yang lebih ringkas siap digunakan!
